# 🧠 The Cognitive Load Balancer

**Live App:** [Click here to try the app on AWS PartyRock]((https://partyrock.aws/u/ArunJyotiChakraborty/gbrxCX4TD/cognitive-load-balancer-ai))

## 📌 Overview
The Cognitive Load Balancer is an AI-driven productivity agent built using Amazon Bedrock via the AWS PartyRock platform. 

Standard chatbots fail at productivity because they lack constraints. This application acts as a "ruthless productivity engineer." It takes a chaotic brain dump of tasks, factors in the user's current energy levels and strict time constraints, and engineers an optimized, micro-step execution plan while explicitly forbidding low-ROI tasks.

## ⚙️ System Architecture & Prompt Engineering

Instead of a single conversational interface, this app utilizes a multi-prompt pipeline to enforce logical reasoning.

### 1. The Ruthless Prioritization Engine
This core widget acts as the filter. It calculates the ROI of tasks against the user's available time.
**System Prompt Used:**
> "You are an elite productivity engineer. Analyze the user's Brain Dump and Urgent Deadlines. The user only has [Available Time Block] hours and their energy is [Current Energy Level]. Execute the following logic: Silently estimate how long each task will take. If the estimated time exceeds the block, you MUST cut tasks. Give me the 'Top 2 High-ROI Tasks'. Finally, identify one task from the brain dump that has a low return on efficiency and explicitly ban the user from doing it today."

### 2. The Micro-Step Execution Plan
This widget takes the output of the Prioritization Engine and reduces friction for the user.
**System Prompt Used:**
> "Look at the Top 2 tasks generated by the Prioritization Engine. Break those tasks down into ridiculously small, frictionless 15-minute micro-steps to overcome task paralysis. Format as a strict checklist."
